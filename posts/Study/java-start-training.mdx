## 훈련

- 변수, 연산자, 조건문, 반복문을 연습

```java
package scanner;

import java.util.Scanner;

public class Scanner1 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("문자열을 입력하세요: ");
        String str = scanner.nextLine(); // 입력을 String으로 가져온다.
        System.out.println("입력한 문자열: " + str);
        
        System.out.print("정수를 입력하세요: ");
        int intValue = scanner.nextInt();
        System.out.println("입력한 정수: " + intValue);
    }
}
```

- `Scanner scanner = new Scanner(System.in);`
    - new로 스캐너를 만든다.
    - Scanner는 System.in을 사용해 사용자의 입력을 편리하게 받도록 도와준다.
    - scanner 변수로 선언해서 사용한다.
- print() vs println()
    - 엔터는 `\n` 이라는 문자를 쓰는 것

### 두수 사이의 값들

```java
숫자1을 입력하세요: 5
숫자2를 입력하세요: 7
두 숫자 사이의 모든 정수:5,6,7
```

```java
import java.util.Scanner;

public class ScannerEx5 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("숫자1을 입력하세요: ");
        int num1 = scanner.nextInt();

        System.out.print("숫자2를 입력하세요: ");
        int num2 = scanner.nextInt();

        if (num1 > num2) {
            int temp = num1;
            num1 = num2;
            num2 = temp;
        }

        System.out.print("두 숫자 사이의 모든 정수:");
        for (int i = num1; i <= num2; i++) {
            System.out.print(i);
            if (i != num2) {
                System.out.print(",");
            }
        }
    }
}
```

### 상품 구매

```java
import java.util.Scanner;

public class ScannerEx9 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int total = 0;

        while(true) {
            System.out.println("1: 상품 입력, 2: 결제, 3: 프로그램 종료");
            int option = scanner.nextInt();

            if (option == 1) {
                scanner.nextLine(); // 엔터 해결을 위해
                System.out.print("상품명을 입력하세요: ");
                String name = scanner.nextLine();
                System.out.print("상품 가격을 입력하세요: ");
                int price = scanner.nextInt();
                System.out.print("구매 수량을 입력하세요: ");
                int quantity = scanner.nextInt();

                total = total + price * quantity;

                System.out.println("상품명: " + name + " 가격: " + price + " 수량: " + quantity + " 합계: " + total);
            } else if (option == 2) {
                System.out.println("총 비용: " + total);
                total = 0;
            } else if (option == 3) {
                System.out.println("프로그램 종료");
                break;
            } else {
                System.out.println("옵션 에러");
            }

        }
    }
}

```

    ```java
    import java.util.Scanner;
    
    public class ScannerEx9_1 {
        public static void main(String[] args) {
            Scanner scanner = new Scanner(System.in);
            int total = 0;
    
            while(true) {
                System.out.println("1: 상품 입력, 2: 결제, 3: 프로그램 종료");
                int option = scanner.nextInt();
    
                switch (option) {
                    case 1:
                        scanner.nextLine(); // 엔터 해결을 위해
                        System.out.print("상품명을 입력하세요: ");
                        String name = scanner.nextLine();
                        System.out.print("상품 가격을 입력하세요: ");
                        int price = scanner.nextInt();
                        System.out.print("구매 수량을 입력하세요: ");
                        int quantity = scanner.nextInt();
    
                        total = total + price * quantity;
    
                        System.out.println("상품명: " + name + " 가격: " + price + " 수량: " + quantity + " 합계: " + total);
                        continue;
                    case 2:
                        System.out.println("총 비용: " + total);
                        total = 0;
                        continue;
                    case 3:
                        System.out.println("프로그램 종료");
                        break;
                    default:
                        System.out.println("옵션 에러");
                }
                break;
            }
        }
    }
    ```

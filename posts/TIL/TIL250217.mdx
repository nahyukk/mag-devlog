## 알고리즘

1978, 1문제 밖에 못품 ㅠㅠ

- **에라토스테네스의 체**
    - 2부터 `N`까지의 숫자 배열 생성
    - 2부터 시작하여 배수를 제거 (`4, 6, 8, ...` 제거)
    - 다음 남아있는 숫자로 반복 (`3`의 배수 제거, `5`의 배수 제거, ...)
    - `√N`까지만 반복하면 됨 (이미 체크된 배수는 자동으로 걸러짐)
        - 어떤 수 `N`이 합성수라면 적어도 하나의 약수가 `√N` 이하에 존재
    - 남아있는 숫자가 소수 🚀
    - 포인트 코드
        
        ```java
        boolean isPrime = true;
        for (int j = 2; j <= Math.sqrt(num); j++) {
            if (num % j == 0) {
                isPrime = false;
                break;
            }
        }
        if(isPrime) count++;
        ```
        
        - 2, 3 은 루트 2, 3이 1.xx 라서 저 for문 자체를 돌지 않고 바로 count가 늘어남.

## `운영체제`

- 실행할 프로그램에 필요한 `자원` 을 할당하고, 프로그램이 올바르게 실행되도록 돕는 특별한 프로그램
    - 자원 - CPU, 메모리, 보조기억장치, 입출력 장치 등 컴퓨터 부품들
- 컴퓨터가 부팅될 때 메모리 내 `커널 영역` 에 따로 적재되어 사용자 영역에 적재된 프로그램에 자원을 할당하고 실행되도록 돕는다.
    - 나머지 영역은 `사용자 영역`
- 지속적인 메모리 자원 관리
    - 사용자 영역의 메모리 주소가 겹치지 않도록 적당한 공간에 적재
    - 실행되지 않는 프로그램을 메모리에서 삭제
    - 공정하게 여러 프로그램에 CPU 자원 할당

### 운영체제 왜 알아야 하는가?

- 하드웨어 상태가 어떤지, 코드가 어떻게 실행 되었는지, 하드웨어 상 어떤 문제가 있었는지 등을 알려 준다. ⇒ 문제 해결의 실마리를 제공
- 하드웨어와 프로그램을 더 깊이 이해할 수 있음
- 운영체제와의 대화 예시 - 오류 메시지

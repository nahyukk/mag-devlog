## 알고리즘

2675, 2920, 3052

- st.nextToken(), st.nextToken() 하면 나눠진 토큰 하나씩 들어간다는 것
- StringBuilder
    
    ```java
    StringBuilder sb = new StringBuilder();
    
    String tone = st.nextToken();
    sb.append(tone);
    
    String scale = sb.toString();
    ```
    
    - `String`으로 바꾸고 싶으면 꼭 `.toString()` 해줘야 한다.
    - 기능들
        
        
        | 기능 | 메서드 |
        | --- | --- |
        | 문자열 추가 | `.append("추가할 문자열")` |
        | 문자열 삽입 | `.insert(위치, "삽입할 문자열")` |
        | 문자열 삭제 | `.delete(시작, 끝)` |
        | 문자열 변경 | `.replace(시작, 끝, "변경할 문자열")` |
        | 문자열 뒤집기 | `.reverse()` |
        | 문자열 길이 확인 | `.length()` |
        | `String`으로 변환 | `.toString()` |
- 배열을 만드는데 동일한 값은 넣지 않음.
    - `Set<Integer> set = new HashSet<>();`
        - `int[] uniqueArray = set.stream().mapToInt(Integer::intValue).toArray();` → set을 배열로 바꿈
    - `List<Integer> list = new ArrayList<>();`
        - `if (!list.contains(num))`중복 검사
- 배열의 길이를 구해서 답을 찾음
    - `list.size()`
    - `list.toArray().length`

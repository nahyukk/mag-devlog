# 블로그 프로젝트 시작 🚀

이 프로젝트는 **개인 블로그를 직접 만드는 과정**을 다룹니다.

## 목표 🎯
- Next.js를 사용하여 블로그를 제작
- Markdown 기반 포스트 작성
- 다크 모드 & 사용자 맞춤 설정 추가

## 사용 기술 🛠
- **Next.js**: 정적 사이트 생성을 위한 프레임워크
- **MDX**: Markdown과 React 컴포넌트 결합
- **Tailwind CSS**: 빠르고 유연한 스타일링

> 🚧 앞으로 기능을 하나씩 추가하며 성장하는 프로젝트가 될 예정입니다!

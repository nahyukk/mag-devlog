## JAVA 형변환

Java에서 **형변환(Type Casting)** 은 데이터 타입을 변경하는 과정입니다.

### 자동 형변환 (Implicit Casting)
자동 형변환은 작은 크기의 데이터 타입이 큰 크기의 데이터 타입으로 변환될 때 발생합니다.

```java
int num = 10;
double doubleNum = num;
```
